## Descripción

La idea principal fue desarrollar una página web para simular un sistema de reserva de asientos en un teatro. El sistema permite visualizar el escenario, ver los asientos disponibles y ocupados, seleccionar los espacios deseados y confirmar una reserva por medio de un formulario.

## Tecnologías utilizadas

- HTML
- CSS
- JavaScript
- Bootstrap

## Funcionalidades

- Visualización del escenario del teatro
- Mapa de asientos
- Distinción entre asientos disponibles, ocupados y seleccionados
- Formulario de reserva
- Confirmación de reserva en pantalla
- Diseño responsivo y estilizado

## Estructura del proyecto

- `index.html` → estructura principal de la página
- `styles.css` → estilos del sistema
- `script.js` → lógica de interacción y reservas

## Objetivo

Aplicar conocimientos de maquetación web, estilos y programación en JavaScript para construir una interfaz funcional y visualmente ordenada.
